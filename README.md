## 기능 구현
  -  좋아요 버튼(좋아요 눌렀을 때 빨간색 하트, 좋아요 개수 늘기)
  -  게시글 이미지 슬라이더(애니메이션) 기능
  -  게시글 저장 기능(아이콘에 색 채우기, 창 띄우기)
  -  댓글 기능(댓글 작성, 삭제, 옵션버튼 눌렀을 때 창띄우기 

  - 서버랑 연결하여 내 아이디, 게시글, 좋아요 수 등등 받아와서 넣어주기
  -  검색 history 구현(최근 검색 항목, 검색기록 지우기 버튼, 모두 지우기 버튼)
